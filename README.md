# Self-Evolving Harness

聚焦增强 OpenClaw 的自主性与可控性。从领域意图理解、自主知识沉淀到自主评测与本地 Agent 架构的端到端自主进化，拒绝在 Skill 层面无序熵增。

**Agent = 大模型 + Harness**

Harness 是大模型之外的一切，目的是解决大模型的可控性与自主进化力缺陷，让 Agent 体验更好。

## 三大模块

- **KIVO** — 认知基础设施（知识、意图、规则、项目、进化）
- **研发流水线** — 执行基础设施（spec → implement → review → deploy → verify）
- **AEO** — Agent 效果运营（度量、归因、质量趋势、成本优化）
